# 📊 Análise de Feedbacks de Clientes Bancários

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio criativo do Bootcamp Bradesco promovido pela DIO (Digital Innovation One).

O objetivo do projeto é simular a atuação de um Analista de Experiência do Cliente (Customer Experience Analyst), realizando a análise de reclamações de clientes de uma instituição financeira para identificar oportunidades de melhoria em produtos, serviços e processos.

A partir de feedbacks públicos de clientes, foram realizadas atividades de classificação, categorização e extração de insights estratégicos, permitindo transformar dados não estruturados em informações relevantes para a tomada de decisão.

---

## 🎯 Objetivos

- Analisar feedbacks reais de clientes.
- Classificar reclamações por categoria.
- Identificar sentimentos predominantes.
- Mapear as principais dores dos clientes.
- Identificar oportunidades de melhoria.
- Gerar insights para aprimorar a experiência do cliente.
- Produzir um relatório executivo com recomendações estratégicas.

---

## 🛠️ Metodologia Utilizada

A análise foi realizada seguindo as etapas abaixo:

### 1. Coleta dos Feedbacks
Utilização de reclamações públicas registradas por clientes.

### 2. Classificação dos Feedbacks
Os relatos foram categorizados em temas como:

- Cobrança e Negativação
- Atendimento
- Cartão de Crédito
- Conta Corrente
- Abertura de Conta
- Segurança
- Suporte ao Cliente

### 3. Análise de Sentimento
Cada feedback foi classificado como:

- Positivo
- Neutro
- Negativo

### 4. Identificação das Principais Dores
Mapeamento dos problemas mais recorrentes enfrentados pelos clientes.

### 5. Geração de Insights
Elaboração de recomendações para melhoria dos processos internos e da experiência do cliente.

### Principais Problemas Identificados

- Cobranças após renegociação de dívidas.
- Permanência indevida em cadastros restritivos.
- Falhas nos canais de atendimento.
- Dificuldade para contestação de compras não reconhecidas.
- Falta de transparência em bloqueios de conta.
- Problemas em processos de abertura de conta.

---

## 💡 Recomendações

### Atendimento ao Cliente
- Melhorar a resolução no primeiro contato.
- Reduzir transferências entre setores.
- Aprimorar os canais digitais de suporte.

### Cobrança e Negativação
- Automatizar a atualização de acordos.
- Garantir a baixa automática das restrições cadastrais.
- Monitorar prazos de regularização.

### Comunicação
- Tornar os processos mais transparentes.
- Informar claramente os motivos de bloqueios e recusas.
- Disponibilizar acompanhamento das solicitações em tempo real.

### Segurança
- Facilitar a contestação de transações suspeitas.
- Criar fluxos de atendimento especializados para fraudes.
